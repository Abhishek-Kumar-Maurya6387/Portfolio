# Abhishek Kumar Maurya Portfolio

## Run

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Build

```bash
npm run build
```

## Important Files

- `src/app/page.jsx` - main portfolio page
- `src/styles/globals.css` - complete styling
- `src/components/sections` - page sections
- `public/images/profile.png` - profile photo
- `public/videos/hero-bg.mp4` - hero background video
- `public/resume/resume.pdf` - resume PDF

## Update Later

- Replace `your.email@gmail.com` in `Hero.jsx` and `Contact.jsx`
- Replace placeholder project links in `Projects.jsx`
